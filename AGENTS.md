# Coding Standards
The environment to use is defined in the root .env file.
Everything in the codebase must be in English language.
For Python, follow PEP 8 guidelines.
For JavaScript/TypeScript, follow Airbnb style guide.
No emojis in the codebase.
Do not create unnecessary documentation or markdown files to report your actions.
Do not create unnecessary scripts, or delete them once they are no longer needed.
When modifying a service, ensure that all related services are also updated accordingly.
